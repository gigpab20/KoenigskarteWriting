\subsubsection{Git}
Git is a VCS that was developed by Linus Torvalds. The of using a VCS is to keep track of different versions of files. Git is the most widely used option, concerning this area of application, due to the fact is it open-source, therefore free, as well as reliable and easy to learn. It gets used in almost every, but not only, development project, not just for tracking the history of files, but also for developing cooperatively making use of it's branching feature. This allows for development while maintaining a stable version on the main branch.
